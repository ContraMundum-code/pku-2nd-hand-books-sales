# 书香北大 · 二手书交易平台
北京大学专属校园二手书交易静态网站，以**教材循环、知识共享、同学互助**为核心，打造安全、高效、便捷的校内书籍流通生态。

---

## 一、产品定位
专为**北京大学校内学生**设计的**一站式二手书交易与知识共享平台**，解决校内书籍交易分散、效率低、体验差的问题，构建集**二手书买卖、图书漂流、社区交流**于一体的绿色学术资源循环体系。

## 二、目标用户
核心用户：北京大学全体本科生、硕士/博士研究生
- 兼具**购书**与**售书**双重需求
- 价格敏感、重视效率、时间碎片化
- 有书籍无偿捐赠/免费领取的公益需求

## 三、需求与痛点
### 1. 核心需求
- 快速找到对应课程/专业的二手教材与参考书
- 低成本、高时效完成书籍买卖与交付
- 闲置书籍便捷处理，回收成本或公益捐赠
- 安全可信、校内闭环的交易环境

### 2. 现存痛点
- 依赖树洞等非专业渠道，**信息杂乱、检索困难、更新不及时**
- 买卖双方需反复私聊沟通，**流程繁琐、效率低**
- 校外平台新书贵、物流慢，**无法满足即时用书需求**
- 缺乏统一管理，**交易状态不透明、易产生纠纷**
- 闲置书籍无规范捐赠渠道，**资源浪费**

## 四、主要功能
1. **首页**：全局导航、搜索入口、热门书籍与动态推荐
2. **购书通道**：多维度筛选（学科/价格/成色）、卡片式列表、书籍详情
3. **售书通道**：一键发布书籍信息、图片上传、订单管理
4. **图书漂流**：书籍免费捐赠、领取、环保贡献统计
5. **论坛社区**：经验分享、求购咨询、交易交流
6. **个人中心**：订单管理、交易记录、收藏、漂流跟踪
7. **系统设置与帮助中心**：账户配置、常见问题解答、人工咨询

## 五、产品亮点
- **校园专属**：仅面向北大师生，身份可信、交易安全
- **流程极简**：发布—筛选—沟通—下单—面交全链路闭环
- **信息标准化**：书籍元数据完整、品相清晰、减少沟通成本
- **公益属性**：图书漂流模块，促进知识共享与绿色环保
- **体验统一**：全局导航+场景化导航，界面一致、易上手
- **纯前端静态**：无后端依赖，可直接本地运行/部署

## 六、当前实现与潜在问题
### 已实现
- 完整前端页面架构（15个核心页面）
- 规范化信息架构、元数据、标签、导航设计
- 基于 Tailwind CSS 的统一视觉风格
- 本地可直接预览的静态网站

### 未完成/潜在问题
- 无后端与数据库，**搜索、登录、订单持久化未实现**
- 图片上传、实时议价、论坛排序/搜索功能缺失
- 无管理员后台，无法审核内容与处理纠纷
- 纯静态页面，**数据无法保存、刷新即丢失**
