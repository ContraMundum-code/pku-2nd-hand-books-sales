# 北京大学二手书交易平台 核心网页说明

（一）首页（首页.html）

该网页文件为 “北京大学二手书交易平台” 的主入口页面，是用户访问平台的首个页面。



1. 页面布局：页面采用流式布局，从上到下依次划分为多个核心区域，包括页眉、核心功能入口区、推荐书籍展示区、平台简介区和页脚。

2. 页眉：页眉部分包含了平台的 logo、全局导航菜单和用户快捷入口（登录 / 注册）。导航菜单提供了购书通道、售书通道、图书漂流、论坛等核心链接，方便用户快速跳转至对应功能模块。

3. 核心功能入口区：该区域位于页眉下方，是页面核心视觉区域，通过图文结合的方式展示平台四大核心功能（购书、售书、图书漂流、论坛交流），每个功能都配有专属图标和简短说明，引导用户快速进入对应页面。

4. 推荐书籍展示区：展示平台内优质待售书籍、热门漂流书籍，采用卡片式布局呈现书籍封面、书名、价格等关键信息，方便用户快速浏览优质资源。

5. 平台简介区：详细介绍北京大学二手书交易平台的设立背景、服务宗旨（便捷师生、资源循环、降低购书成本）和使用须知，帮助新用户快速了解平台价值和使用规则。

6. 页脚：页脚部分包含了平台版权信息、联系方式、帮助中心入口和法律声明，为用户提供必要的平台辅助信息和问题反馈渠道。

   该网页采用简洁实用的设计风格，贴合校园二手交易的使用场景，导航清晰，便于师生快速上手操作。

**ii. 代码运行说明：**

**项目概况**



* 静态多页网站，中文 HTML 文件，无后端、数据库或 API。

* 样式通过 CDN 引入 Tailwind CSS，无构建步骤。

* 图标使用 Iconify（CDN），图片包括本地资源（logo.png）与外部引用。

* 页面跳转通过相对路径链接（如 首页.html、购书通道.html），保持文件名与链接一致。

  &#x20;

  **编辑与运行**

* 每页为完整 HTML 文件（head + body），修改共享布局（头部/底部）需在所有页面同步。

* Tailwind 主题颜色在页面内通过内联脚本配置，修改需同步所有页面。

* 脚本为内联或 \<script> 标签引入，无模块化打包。

* 本地链接使用中文文件名，重命名文件前请更新所有相关链接。

  &#x20;

  **本地测试**

1. 直接打开 HTML 文件可预览，但建议使用静态服务器以支持相对路径。

2. 启动静态服务器（macOS / zsh）：

   python3 -m http.server 8000

   访问 [http://localhost:8000/首页.html](http://localhost:8000/%E9%A6%96%E9%A1%B5.html) 查看页面。

   &#x20;

   **注意事项**

* 修改 header/footer 等共享组件前，搜索并更新所有页面。

* 保持 logo.png 路径不变，避免加载失败。

* 导航相关文件（首页.html、论坛.html等）重命名需同步更新链接。

  &#x20;

  **未来优化方向**

* 提取共享组件（头部/底部）为独立文件（如 inc/header.html），减少重复修改。

* 若需要 Tailwind JIT 或 JS 模块化，可配置对应工具链（package.json 等）。
